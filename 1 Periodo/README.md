# 📘 Período 1 – Introdução à Programação
# 1º Semestre de 2024

Neste período, foi desenvolvido um projeto simples com foco nos **fundamentos da lógica de programação**, utilizando a linguagem **C**.

---

## 🎮 Projeto: Pedra, Papel e Tesoura

Implementação do jogo **Pedra, Papel e Tesoura** em C, executado no terminal.

### 🧠 Objetivo
Praticar conceitos básicos de programação, como:
- Entrada e saída de dados
- Estruturas condicionais
- Uso de variáveis
- Lógica de decisão

---

## 🛠️ Conceitos Utilizados

- Linguagem C
- `scanf` e `printf`
- `if / else`
- Geração de jogada do computador
- Comparação de resultados
